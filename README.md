# BurpSuite BChecks Collection

Made with Dia. 

A custom collection of BChecks for Burp Suite Professional and Burp Suite DAST. This repository contains custom scan checks that you can use to identify specific vulnerabilities, misconfigurations, and other patterns during web application security testing.

## How to Use BChecks

To import and use these checks in your Burp Suite Professional instance:

1. **Download the BChecks**: Clone this repository or download the individual `.bcheck` files you wish to use.
2. **Open Burp Suite**: Launch Burp Suite Professional.
3. **Navigate to BChecks**: Go to the **Extensions** tab, then select the **BChecks** sub-tab.
4. **Import**: Click the **Import** button and select the `.bcheck` files you downloaded.
5. **Enable Checks**: Ensure the checks are enabled so they run during your active or passive scans.
